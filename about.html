根據教育部課程規劃，台灣中小學生均有參與科展(science fair) 或撰寫小論文的需求，但普遍來說圖書館員在中小學生科展及小論文的參與和影響甚少。

此人文科展/小論文資訊平台面相為高中生

進行科展或小論文研究需要具備良好的科學素養，包括：

  • 提出有挑戰性和實際意義的問題。
  • 掌握資料搜集和分析的方法。
  • 能夠進行系統性的實驗或觀察。
  • 具備清晰、結構化的文獻寫作能力。

國高中科學展覽研究指南


1. 選題和初步研究

與指導老師討論，選擇一個感興趣的問題，確定研究方向。

  • 圖書館資源管理系統（如 Destiny Library Manager）
  • Google Scholar：用於查找學術文章和資料。
  • PubMed（對生物醫學領域特別有用）

2. 研究計劃制定

繪製研究計劃，包括所需材料、實驗設計和時間表。

  • Microsoft Word 或 Google Docs：用於撰寫和格式化研究計劃。
  • Trello 或 Asana：項目管理工具，幫助組織研究任務和時間線。

3. 數據收集

在實驗室進行實驗，按照計劃收集數據。

  • LabArchives：一個電子實驗室筆記本，用於記錄和存儲實驗數據。
  • Google Sheets 或 Microsoft Excel：用於數據錄入和初步分析。

4. 數據分析

使用統計方法分析收集到的數據。

  • SPSS：強大的統計分析軟體，適合進行複雜的數據分析。
  • JMP：一款強大的統計軟件，提供動態數據可視化。

5. 研究報告撰寫

撰寫包括研究介紹、方法、結果和結論的完整報告。

  • Microsoft Word 或 Google Docs
  • Zotero 或 EndNote：參考文獻管理軟體，幫助組織和引用研究文獻。

6. 展示準備

創建展示板和/或演示文稿以展示研究結果。

  • Microsoft PowerPoint 或 Google Slides：製作演示文稿。
  • Canva：一個易於使用的圖形設計工具，適合製作視覺吸引人的展示板。

7. 參與科展和反饋

在科展中展示研究，收集評審和觀眾的反饋。

  • Google Forms：用於收集和組織觀眾反饋。
